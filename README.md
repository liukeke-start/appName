### 1.安装gulp
```
npm install gulp -g
npm install gulp --save-dev 
```
### 2.本地初始化
```
npm install
```
### 3.启动项目
```
gulp run
```
### 4.构建
```
gulp 
```
### 5.构建生成版本
```
gulp build
```
